# rsgyc-forms
Form with parsley validation and stripe payment

Final website url: [https://www.rsgyc.ie/berthing-form/](https://www.rsgyc.ie/berthing-form/)


### Installation

You need to install JS, PHP and CSS libriaries:

```sh
$ npm install
$ npm update
$ bower update
$ grunt
```
